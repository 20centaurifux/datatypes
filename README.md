# datatypes - README

This library provides some datatypes you might miss in C. The following
datatypes are available:

* hashtable
* red black tree
* singly-list
* doubly-linked list
* queue
* stack
* asynchronous (message) queue (requires POSIX threads!)

To generate the code documentation please run Doxygen[^1].
